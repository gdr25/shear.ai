# 💇‍♀️ AI Salon Style Preview Platform

An AI-powered web application that allows users to **preview hairstyles, hair colors, and beard styles on their own face before booking a salon appointment**.

> 🚀 Reduce cancellations. Increase confidence. Transform salon experiences.

---

## 🌟 Overview

Choosing a hairstyle is risky for customers. This platform eliminates uncertainty by enabling users to:

* Upload their photo or use a live camera
* Try different hairstyles and colors in real-time
* Compare before vs after looks
* Book appointments with confidence

---

## 🎯 Key Features

### 🧠 AI Style Preview

* Face detection and mapping
* Realistic hairstyle overlay
* Hair color customization

### 🎥 Live Try-On (AR Mode)

* Real-time camera-based preview
* Dynamic hair rendering based on movement

### 💡 Smart Recommendations

* AI suggests styles based on:

  * Face shape
  * Hair density
  * Trends

### 🔄 Before vs After Comparison

* Interactive slider view
* Helps users decide faster

### 💾 Save & Share

* Save favorite looks
* Share via social media

### 📅 Instant Booking

* Book directly from selected style
* Send preview to stylist

### 👤 User Dashboard

* Saved styles
* Booking history
* Personalized suggestions

## 🧱 Tech Stack

### Frontend

* React.js / Next.js
* Tailwind CSS
* Three.js (for 3D/AR rendering)

### Backend

* Node.js / Express OR Python (FastAPI)

### AI / ML

* MediaPipe (face detection)
* OpenCV (image processing)
* Stable Diffusion / ControlNet (style rendering)
* InsightFace (face mapping)

### Cloud & Infra

* AWS / GCP
* GPU-enabled servers for AI processing

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/ai-salon-preview.git
cd ai-salon-preview
```

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

#### Backend

```bash
cd backend
npm install
npm run start
```

---

## 🔌 Environment Variables

Create a `.env` file in the backend:

```
PORT=5000
AI_MODEL_API_KEY=your_api_key
CLOUD_STORAGE_KEY=your_key
```

## 📸 How It Works

1. User uploads image or opens camera
2. AI detects face structure
3. Hairstyles are applied using ML models
4. User compares styles
5. User books appointment
