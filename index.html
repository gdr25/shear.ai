🚀 Deploy Your Website
This is a single HTML file — no server, no coding needed. Pick any method below to go live in minutes.
1
Download this file first

Click the download/save button in Claude to save salon-ai.html to your computer.

2
⚡ Netlify Drop — Easiest (Free, No Account)

Drag and drop your HTML file — your site is live instantly.

1. Visit: app.netlify.com/drop
2. Drag your salon-ai.html onto the page
3. ✅ Live at yoursite.netlify.app in 10 seconds
Open Netlify Drop →
3
🌐 Vercel — Best (Free + Custom Domain)

Best for a permanent site. Use their website (no coding needed):

1. Go to: vercel.com → Sign Up (free)
2. Click "Add New → Project"
3. Upload your salon-ai.html
4. ✅ Live at your-project.vercel.app

Or via terminal: npx vercel --name shear-ai

4
📄 GitHub Pages (Free)

Rename file to index.html, create a GitHub repo, then:

Settings → Pages → Source: main branch → Save
✅ Live at username.github.io/repo-name
5
🔥 Add Real AI (Next Level)

To add real face detection and AI hair rendering:

• Face detection: face-api.js (free, runs in browser)
• Hair segmentation: MediaPipe (Google, free)
• AI renders: Replicate.com API (Stable Diffusion)
• Booking backend: Supabase (free tier)
• Payments: Razorpay or Stripe
6
🌏 Custom Domain (Optional)
Buy domain: GoDaddy / Namecheap (~₹800/year)
Point DNS to Netlify/Vercel → done in 10 mins
